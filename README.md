# Mutti's Bakery Production System

This is a professional-grade backend production system designed for Mutti's Bakery. The system manages recipes, provides scaling calculations with non-linear constraints, handles authentication, and includes an emergency "Break-Glass" security protocol.

## Project Structure

- `src/`: Python source files.
- `data/`: JSON storage for recipes and unit conversions.
- `logs/`: Application audit logs.
